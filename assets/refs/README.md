# Referências fixas — Whisk

Imagens usadas como referência em todas as campanhas estáticas do Vivr.
Coloque os arquivos aqui e aponte os caminhos em `src/config.ts`.

## Estrutura esperada

| Arquivo | Categoria Whisk | O que deve ser |
|---|---|---|
| `subject.png` | Subject | Personagem principal — adulto, estilo 3D cartoon, identidade visual do Vivr |
| `scene.webp` | Scene | Ambiente de referência — ex: café moderno, espaço urbano |
| `style.webp` | Style | Estética visual — render 3D, gradiente brand, iluminação suave |

## Como usar

Os caminhos são lidos automaticamente de `src/config.ts`.
Para trocar uma referência, substitua o arquivo e mantenha o mesmo nome.
