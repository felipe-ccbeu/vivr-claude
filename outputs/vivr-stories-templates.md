## 🎨 Design System Shared (All Templates)

### Typography
- **Headline Font:** Nunito 800 (extra-bold)
- **Body Font:** Nunito 500 (regular)
- **Line Height:** 1.2 para headlines; 1.5 para body

### Colors (Brand Palette)
- **Primary Gradient:** `#89c7fe → #8bfbd1 → #ae90fb → #f599b5 → #fdd38a` (135deg)
- **Dark Background:** `#0d0d0d`, `#1a1030`, `#0d0d00`
- **Text:** Always white or light gray on gradients

### Visual Effects
- **Shadows:** `0 4px 20px rgba(0,0,0,0.3)` for depth
- **Blur:** `backdrop-filter: blur(10px)` for glass cards
- **Glow:** `filter: drop-shadow(0 0 12px rgba(249,112,64,0.45))`

### Story Specs (540×960px)
- **Safe Zone:** 28px padding all sides
- **Header Area:** Top 120px (hook + small context)
- **Main Content:** 120px - 800px (image, text, visual hierarchy)
- **CTA Zone:** Bottom 160px (button + sub-label)
- **Margin:** 28px from edges always

### Interactive Elements
- **Button Padding:** 14px 30px (standard)
- **Border Radius:** 100px (pill buttons) or 8px (cards)
- **CTA Colors:** Gradient primary or white text on gradient
- **Hover:** Scale 1.05 + opacity shift

### Animations
- **Duration:** 300-400ms for fade-in/slide
- **Easing:** ease-out for entrance, ease-in-out for continuous
- **Parallax:** 20-30px offset between layers

## 📋 Implementation Notes

### For Each Template:

1. **Start with Story Canvas (540×960px)**
   ```html
   <div class="story-container w-screen max-w-[540px] h-[960px]">
   ```

2. **Gradient Backgrounds**
   - Use CSS `linear-gradient(135deg, color1, color2, ...)`
   - Layer with `::before` for overlays

3. **Typography Hierarchy**
   - Headline: 44-50px, font-weight-800, white
   - Description: 12-14px, font-weight-500, white/70%
   - CTA: 14px, font-weight-800, white

4. **Images**
   - Use `object-position: center 30%` to focus on faces
   - Always apply overlay gradient for text legibility
   - Aspect ratio 540×600px for image areas

5. **CTA Placement**
   - Bottom 160px reserved
   - Button center, sub-label below
   - Always have contrast against background

6. **Animation Order**
   1. Background fade-in (0-200ms)
   2. Image/character entrance (200-500ms)
   3. Text slides/fades (300-600ms)
   4. CTA button pulse (800-1200ms)
